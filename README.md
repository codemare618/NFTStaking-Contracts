# NFTStaking Contract
An NFT Staking pool which airdrops ERC20 (ASHToken) as Reward

## Requirements

- Node.js  v10.0 or higher


## Install dependencies

```shell
$npm install
```


## Testing

- Run shared hardhat node

```shell
$npx hardhat node
```

- Test ERC20Token

```shell
$npm run test-token
```

- Test Pool
```shell
$npm run test-new-
```
